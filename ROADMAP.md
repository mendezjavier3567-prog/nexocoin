# Nexocoin Roadmap

This document outlines the current state and forward direction of the Nexocoin project.
It is provided for transparency and alignment purposes only and does not constitute
a commitment to specific timelines.

---

## Current Status

Nexocoin is an operational, independent base-layer blockchain network.

The following components are implemented and functional:

- Genesis block and base chain structure
- Cryptographic security (public/private keys, signatures, verification)
- Immutable transaction model
- Peer-to-peer network (NexoNet)
- Multi-node operation
- Hybrid consensus mechanism (PoW + PoS)
- Defined economic model and fixed supply
- Persistent mining and validation
- Official CLI wallet
- Real transactions and balances
- Native coin (NXC), not a token

At this stage, Nexocoin exists as a usable digital currency system with a live network
and enforced protocol rules.

---

## Near-Term Evolution

The protocol is designed to evolve conservatively without disrupting the existing network.

Planned technical evolution areas include:

- Refinement of node communication and peer state handling
- Improved logging, error handling, and operational robustness
- Gradual expansion of developer-facing tooling

These changes are incremental improvements to an already functioning system.

---

## Future Capabilities

Future protocol layers are designed but intentionally introduced only after stability
and correctness are fully validated.

These include:

- Native smart contract execution (Nexo VM / Nexo Script)
- Advanced staking and DeFi primitives
- Programmable tokens and on-chain governance
- Developer SDKs and public APIs
- Extended explorer and ecosystem tooling

No activation dates are defined for these capabilities.

---

## Guiding Principles

Nexocoin prioritizes:

- Security over speed
- Predictability over experimentation
- Long-term sustainability over short-term growth
- Backward compatibility and protocol stability

All protocol evolution follows these principles.
