# Nexocoin Code of Conduct

## Purpose

The Nexocoin project is committed to providing a professional, respectful,
and secure environment for collaboration, discussion, and contribution.

This Code of Conduct establishes clear expectations for behavior in all
project-related spaces, including repositories, discussions, and community
interactions.

## Expected Behavior

Participants are expected to:

- Act professionally and respectfully at all times
- Engage in constructive technical discussion
- Respect differing viewpoints and experience levels
- Focus on protocol quality, security, and long-term stability
- Accept responsibility for mistakes and correct them when identified

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or personal attacks
- Aggressive, abusive, or inflammatory language
- Misinformation presented as official protocol behavior
- Attempts to misrepresent the project or its governance
- Unauthorized claims of official representation

## Scope

This Code of Conduct applies to all official Nexocoin spaces, including:

- GitHub repositories
- Issue trackers
- Pull requests
- Documentation discussions
- Any official project communication channels

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Violations may result in actions including warnings, content removal,
or restriction of participation, depending on severity.

## Reporting

Security-related issues must be reported according to SECURITY.md.

Behavioral concerns may be reported privately to the project maintainers.

## Governance

This Code of Conduct may be updated through formal project governance
processes to ensure continued alignment with Nexocoin’s values.
