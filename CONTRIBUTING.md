# Contributing to Nexocoin

Thank you for your interest in contributing to Nexocoin.

## Contribution Scope

Nexocoin protocol version v0.1.0 is frozen.
No backward-incompatible changes will be accepted for this version.

All contributions must target future versions (v0.2.0 and above).

## What Can Be Contributed

- Documentation improvements
- Clarifications and technical corrections
- Tooling, testing, and reference implementations
- Proposals for future protocol versions

## What Will Not Be Accepted

- Changes to consensus rules of v0.1.0
- Changes to monetary supply of v0.1.0
- Breaking changes to protocol definitions

## Process

1. Open an issue describing the proposal
2. Discuss the change before submitting code
3. Submit a pull request targeting a future version branch

All contributions are reviewed manually.

## Code of Conduct

Be respectful and technical.  
This project values clarity, precision, and long-term stability.
