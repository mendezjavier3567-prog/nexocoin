# Contributing to Nexocoin

Thank you for your interest in contributing to the Nexocoin project.

Nexocoin is a security-focused, base-layer blockchain protocol designed with a conservative, long-term approach. All contributions are expected to align with these principles.

---

## Scope of Contributions

At the current stage, Nexocoin primarily accepts contributions related to:

- Documentation improvements
- Technical reviews and clarifications
- Security analysis and responsible disclosure
- Tooling, testing, and developer infrastructure (by prior coordination)

Protocol-level changes are carefully reviewed and not accepted without prior discussion.

---

## Contribution Process

1. Open an issue describing the proposed change or improvement.
2. Clearly explain the motivation, scope, and impact.
3. Wait for project maintainers to review and respond.
4. If approved, submit changes via a pull request.

All pull requests must be clear, minimal, and well-documented.

---

## Code and Documentation Standards

- Clarity and correctness are prioritized over speed.
- Backward compatibility must be preserved whenever possible.
- All changes should be auditable and easy to review.
- Documentation must remain precise and technically accurate.

---

## Security

If you discover a potential security issue, **do not open a public issue**.

Please follow the responsible disclosure process described in the `SECURITY.md` file.

---

## Communication

All communication should remain professional, technical, and respectful.

Speculation, hype, or price-related discussion is outside the scope of this repository.

---

## Final Notes

By contributing to this repository, you agree that all submissions may be reviewed, modified, or declined to preserve the long-term integrity and security of the Nexocoin network.

Thank you for helping improve Nexocoin.
